Oblivion — это ориентированное на безопасность Android-приложение, которое позволяет пользователю заблокировать и очистить устройство при срабатывании триггера.

Приложение предназначено для ситуаций повышенного риска, таких как потеря устройства, принудительная разблокировка или временная утрата физического контроля.

Методы срабатывания:
    * Тревожная кнопка (плитка в быстрых настройках)
    * Удалённое сообщение, содержащее секретный код
    * Срабатывание, если устройство не было разблокировано в течение настраиваемого периода времени

Действия:
    * Выполнение сброса к заводским настройкам (полное удаление данных)

Управление устройством реализовано с использованием API администрирования устройств Android. Приложение полностью поддерживает GrapheneOS.

Примечания по безопасности:
    * Только зашифрованные устройства (защищённые паролем) могут гарантировать невозможность восстановления данных
    * Приложение не работает в безопасном режиме
